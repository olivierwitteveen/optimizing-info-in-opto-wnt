# Processed data

Processed experimental data. 

Single-cell TopFlash data in `g.pkl` and the corresponding opto-Wnt duration in `t.pkl`. 

Raw data is available at: https://zenodo.org/records/18601164. 

The file `optimal_priors.pkl` contains the optimal priors as a function of the effective noise.

`spatial_data/` containsed processed experimental data from a single-well repeat experiment for 12.5 hours opto-Wnt. Contains TopFlash expression, `g.npy`, and the associated xy-coordinates `coordinates.npy`.
