# Optimizing information transmission in optogenetic Wnt signaling

Work performed by Bauer Group at TU Delft and Max Wilson lab at UCSB.

# Reference
This repository contains code and data accompanying the paper:

O. Witteveen, S.J. Rosen, R.S. Lach, M.Z. Wilson, M. Bauer, Optimizing information transmission in optogenetic Wnt signaling, Physical Review Research (2026). Available at: https://link.aps.org/doi/10.1103/f7qj-f7qy

## Overview

The code and processed experimental data needed to reproduce the figures are contained in this repository. 

### Folders:
- `figures/`: all figures. Can be generated by `figure_notebook.ipynb`.
- `processed_data/`: processed experimental data. Single-cell TopFlash data in `g.pkl` and the corresponding on-time condition in `t.pkl`. The file `optimal_priors.pkl` contains the optimal priors as a function of the effective noise. 
- `fine_tuning/`: numerical results used to plot Figs. 6 and 9. Mutual information as a function of peak positions in the optimal prior. 

### Notebooks
- `figure_notebook.ipynb`: code to generate all figures. 
- `optimal_prior.ipynb`: standalone notebook with an example for how to implement the Blahut-Arimoto algorithm and its adaptation for discrete priors. 

### Scripts
- `blahut.py`: script containing a standard Blahut-Arimoto implementation and tools to implement an adapted version for discrete priors.
